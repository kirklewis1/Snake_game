Snake Game in Python
A classic snake game implemented in Python using Pygame library.

Requirements
Python 3.x
Pygame
How to run
Clone the repository.
bash

git clone https://github.com/[username]/python-snake-game.git
Install Pygame.

pip install pygame
Run the game.

python snake.py

How to play
Use arrow keys to control the snake and eat the food to grow longer.
Avoid hitting the walls or yourself, or the game will end.

Features
Classic snake gameplay
Dynamic window size
Game over screen
High score tracking
